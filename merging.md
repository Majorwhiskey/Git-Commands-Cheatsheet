# 🔀 Merging & Rebase

## git merge

Combine another branch into current branch.

```bash
git merge feature
```

## git rebase

Reapply commits on top of another branch.

```bash
git rebase main
```

## git merge --abort

Cancel a merge in case of conflicts.

```bash
git merge --abort
```

## git rebase --abort

Stop an ongoing rebase process.

```bash
git rebase --abort
```
