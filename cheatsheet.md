# ⚡ Quick Git Cheatsheet

## Daily Workflow

```bash
git status
git add .
git commit -m "msg"
git pull
git push
```

## Branching

```bash
git checkout -b feature
git merge feature
```

## Undo

```bash
git restore file
git reset HEAD~1
```

## Advanced

```bash
git stash
git cherry-pick <id>
git reflog
```
