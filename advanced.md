# 🔥 Advanced Commands

## git stash

Temporarily save uncommitted changes.

```bash
git stash
git stash pop
```

## git cherry-pick

Apply a specific commit to current branch.

```bash
git cherry-pick <commit_id>
```

## git reflog

View history of HEAD movements.

```bash
git reflog
```

## git rebase -i

Interactively edit commit history.

```bash
git rebase -i HEAD~3
```

## git clean

Remove untracked files.

```bash
git clean -fd
```
